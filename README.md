### Mission

## Getting Started

<!-- Prerequisites -->
### Prerequisites

Before you begin, ensure you have the following installed:

* Python 3.8 or higher
  ```sh
  python --version
  ```
* pip (Python package manager)
  ```sh
  pip --version
  ```
* Virtual environment (recommended)
  ```sh
  pip install virtualenv
  ```

<!-- Installation -->
### Installation

1. Clone the repository
   ```bash
   git clone https://github.com/HoussemEddinChaouach-eniso/Tumor-Detection.git
   ```

2. Navigate to the project directory
   ```bash
   cd Tumor-Detection
   ```

3. Create a virtual environment
   ```bash
   python -m venv venv
   ```

4. Activate the virtual environment
   
   **Windows:**
   ```bash
   venv\Scripts\activate
   ```
   
   **macOS/Linux:**
   ```bash
   source venv/bin/activate
   ```

5. Install required dependencies
   ```bash
   pip install -r requirements.txt
   ```

<!-- Environment Setup -->
### Environment Setup

Create a `.env` file in the project root directory (if needed):

```env
# Flask Configuration
FLASK_APP=app.py
FLASK_ENV=development
SECRET_KEY=your_secret_key_here

# Model Configuration
MODEL_PATH=models/tumor_detection_model.h5
UPLOAD_FOLDER=uploads/
ALLOWED_EXTENSIONS=png,jpg,jpeg,dcm

# 3D Model Export
STL_EXPORT_PATH=exports/stl/
```

<!-- Run Locally -->
### Run Locally

1. Ensure your virtual environment is activated

2. Run the Flask application
   ```bash
   python app.py
   ```
   or
   ```bash
   flask run
   ```

3. Open your browser and navigate to:
   ```
   http://localhost:5000
   ```

4. Upload an MRI scan and start detecting tumors!

<!-- Usage -->
## Usage

### For Medical Professionals

1. **Upload MRI Scan** - Select and upload brain MRI images (DICOM, PNG, JPG formats supported)
2. **AI Analysis** - The system automatically processes the scan using the trained model
3. **View Results** - Review detection results, tumor classification, and confidence scores
4. **3D Visualization** - Explore the interactive 3D reconstruction of detected tumors
5. **Export STL** - Download STL files for 3D printing or further analysis
6. **Generate Report** - Create comprehensive diagnostic reports

### For Developers

```bash
# Train the model with your dataset
python train_model.py --dataset path/to/dataset --epochs 50

# Evaluate model performance
python evaluate_model.py --model models/tumor_detection_model.h5

# Generate predictions
python predict.py --image path/to/mri_scan.jpg

# Export 3D model
python generate_3d_model.py --input predictions/tumor_mask.npy --output tumor_model.stl
```

## Model Performance

### Accuracy Metrics

| Metric | Score |
|--------|-------|
| Overall Accuracy | **92.3%** |
| Precision | **91.8%** |
| Recall | **93.1%** |
| F1-Score | **92.4%** |

### Training Details

- **Dataset Size**: Multiple brain MRI datasets with thousands of samples
- **Validation Method**: K-Fold Cross-Validation (k=5)
- **Data Augmentation**: Applied to increase dataset diversity
- **Class Balancing**: SMOTE and undersampling techniques
- **Training Time**: Optimized for efficient convergence
- **Architecture**: Convolutional Neural Network (CNN) with multiple layers

### Classification Categories

The model can detect and classify:
- Glioma
- Meningioma
- Pituitary Tumor
- No Tumor (Healthy Brain)
